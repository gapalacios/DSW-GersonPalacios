Diseño de sitios web según conceptos de Flex Box y CSS Grid



Propósito:


Este sitio tiene como propósito dar información concerniente al diseño de sitios web
orientado a la aplicación de Flex Box y CSS Grid, donde encontrara información relevante
donde aprenderemos que son, cuál es su función, sus principales características, ventajas
y desventajas al realizar su aplicación.




Sección Recursos Digitales:


Adicionalmente en el menu de Recursos Digitales podemos encontrar viedos explicativos donde
donde los internautas podran comprender de una manera practica y sencilla los diferentes conceptos
que normalmente son utilizados en el diseño de paginas web.
